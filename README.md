Real-Time Interactive Cancer Risk Prediction Using Machine Learning with Self-Training Feedback Loop

Overview
This repository contains the official implementation of the research work presented at the IEEE International Conference on Advances in Nanotechnology (ICONAT 2025).   The project develops a real-time cancer risk prediction system using machine learning and an adaptive self-training feedback loop that continuously improves model performance based on new user inputs.

Features
- Real-time cancer risk inference using tabular clinical/lifestyle data  
- Self-training feedback mechanism for incremental model improvement  
- Ensemble ML models including Random Forest, XGBoost, and DNN  
- Feature preprocessing, normalization, and imbalance handling  
- Evaluation metrics: Precision, Recall, F1-Score, ROC-AUC  
- Interactive model demonstration (optional notebook/UI)

Machine Learning Models Used
- Random Forest  
- XGBoost  
- Logistic Regression  
- Deep Neural Networks (DNN)  
- Self-training wrapper for semi-supervised adaptation
